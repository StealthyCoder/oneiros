# Findings

Observations from actually running the Oneiros pipeline, as distinct from
[`docs/todos/`](../todos/README.md), which tracks build work. One file per
run or per analysis pass, named by date.

Run output itself (`output/`) is gitignored - these notes record what was
found, not the underlying images/narratives, so a note here is only as
reproducible as its own quoted numbers and examples.
